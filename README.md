## Technologies Used
- C++11
- Qt 5.15.2 LTS
- CMake 3.5

## Build
- `cmake -S /path/to/cfp_simulator/src -B /path/to/build-dir -D CMAKE_BUILD_TYPE=Release` 
- `cmake --build /path/to/build-dir`
